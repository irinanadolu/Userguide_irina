Contributing a Custom Preferences Page

There is a specific plugin extension type that can be used to contribute a custom preferences page to the Preferences dialog box in Oxygen XML Editor. An example of how such a page is implemented can be found in this sample plugin: https://github.com/oxygenxml/oxygen-dita-prolog-updater/blob/master/src/main/java/com/oxygenxml/prolog/updater/view/PrologOptionPage.java.
