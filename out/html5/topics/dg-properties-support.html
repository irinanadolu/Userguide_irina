Supported CSS Properties

Oxygen XML Editor validates all CSS 2.1 properties, but does not render aural and paged categories properties in Author mode, as well as some of the values of the visual category that are listed below under the Ignored Values column. For the Oxygen XML Editor-specific (extension) CSS properties, see CSS Extensions.
