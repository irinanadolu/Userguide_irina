Supported Processors

The DITA-OT CSS-based PDF Publishing Plugin supports the following CSS processors:
  • Oxygen PDF Chemistry - This is recommended processor because the built-in CSS files were fine-tuned for this processor. For example, metadata extraction only functions with this processor. If the plugin is started from an Oxygen XML Editor/Author distribution, a Chemistry installation is not needed.
  • Prince XML - A commercial product, available at: https://www.princexml.com/.
  • Antenna House - A commercial product, available at: https://www.antennahouse.com/antenna1/formatter/.
