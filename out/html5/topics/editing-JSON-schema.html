Editing JSON Schema Documents

The purpose of the JSON schema is to define the legal properties and values of a JSON document to keep it valid and well formed. When editing JSON Schema, all of the same features in the JSON editor are available.

Text Mode Editor

When editing JSON Schema documents in Text editing mode, the usual text editing actions are available, along with other editor-specific actions, including:
  • Search and Find/Replace
  • Drag and Drop
  • Validation
  • Format and Indent (Pretty Print)

Author Visual Editor

It is also possible to edit JSON files in the visual Author editing mode. Oxygen XML Editor uses a specific CSS for rendering JSON Schema documents in Author mode and you have access to the various features and actions that are available when editing XML documents in Author mode.

New Document Templates

Oxygen XML Editor includes a new document template to help you get started creating a JSON Schema document. The template is called JSON Schema and it can be found in the New Document folder in the New document wizard. You can also customize your own JSON Schema templates and specify other versions (draft 04, 06, or 07).
Tip: You can experiment with a sample of a JSON Schema file available at: [OXYGEN-INSTALL-DIR]/samples/json/personal-schema.json.
