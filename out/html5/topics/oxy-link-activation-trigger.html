Link Navigation: -oxy-link-activation-trigger Property

Used to specify how hyperlinks are handled in Author mode.

The -oxy-link-activation-trigger property is used to specify when hyperlinks are clickable in Author mode. This is helpful for those who are used to the hyperlink activation procedure in other applications (for example, apps that use Ctrl+Click (Command+Click on OS X) to activate hyperlinks.

The possible values are:
  • click - Hyperlinks are opened when a user mouse-clicks the link icon or text.
  • modifier-click - Hyperlinks are opened when a user holds down Ctrl (Command on OS X) and mouse-clicks the link icon or text.
  • auto - The hyperlink strategy is determined automatically, depending on the context.
  • inherit - The value is inherited from the parent element.
