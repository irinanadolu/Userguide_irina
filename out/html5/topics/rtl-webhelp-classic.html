How to Activate Support for Right-to-Left (RTL) Languages

To activate support for RTL (right-to-left) languages in WebHelp output, set the @xml:lang attribute with the corresponding attribute value:
  • ar-eg - Arabic
  • he-il - Hebrew
  • ur-pk - Urdu
