Customizing the Editing Experience for a Framework

You can customize the editing experience in Author mode for you and any other user who shares the same framework. This includes the ability to configure actions, menus, toolbars, icons, structure insertion shortcuts, and content completion proposals specifically for a particular framework (document type). Advanced users who are familiar with API development can also create custom Author mode operations for a particular framework.
