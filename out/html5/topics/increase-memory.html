How to Increase the Amount of Available Memory

Determining how to increase the amount of memory that is allocated to Oxygen XML Editor depends on how you launch the application.
  • Windows/Linux Application Launcher - If you start Oxygen XML Editor using the default startup launcher that was created during a Windows or Linux installation, see Increasing the amount of memory that Oxygen XML Editor uses on Windows and Linux.
  • Mac OS X Application Launcher - If you start Oxygen XML Editor using the default startup launcher that was created during a Mac OS X installation, see Increasing the amount of memory that Oxygen XML Editor uses on OS X.
  • Command-Line Script - If you start Oxygen XML Editor using a command-line script, see Setting Parameters in the Command-Line Scripts.
  • Custom Startup Parameters File - You can also create your own custom startup parameters file and increase the memory using this file. For more information, see Creating Custom Startup Parameters File.
