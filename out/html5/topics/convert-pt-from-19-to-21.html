Convert Version 19 (and Older) Publishing Templates to Version 21

If you have a custom template that was created in Oxygen XML Editor version 19.1 or older and you want to convert it to be compatible with Oxygen XML Editor version 21.0 or 21.1, you need to apply two conversion procedures:
  1. Convert the old template to a version 20 Publishing Template by using this procedure: Convert Version 19 (and Older) Publishing Templates to Version 20.
  2. Convert that version 20 Publishing Template to version 21 by using this procedure: Convert Version 20 Publishing Templates to Version 21.

Result: The converted Publishing Template can now be used in version 21.0 or 21.1.
