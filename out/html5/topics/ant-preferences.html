Ant Preferences

To set Ant preferences, open the Preferences dialog box (Options > Preferences) and go to XML > Ant. This panel allows you to choose the directory containing the Apache Ant libraries (the so-called Ant Home) that Oxygen XML Editor uses to handle Ant build files.

There are two options available:
  • Built-in - the path to the Ant distribution that comes bundled with Oxygen XML Editor installation kit.
  • Custom - the path to an Ant distribution of your choice.
