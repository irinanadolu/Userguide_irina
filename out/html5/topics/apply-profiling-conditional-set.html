Applying Profiling Condition Sets

All defined Profiling Condition Sets are available as shortcuts in the Profiling / Conditional Text toolbar menu. Select a menu entry to apply the condition set. The filtered content is then grayed-out in the Author mode and Outline view. An element is filtered-out when one of its attributes is part of the condition set and its value does not match any of the values covered by the condition set.
