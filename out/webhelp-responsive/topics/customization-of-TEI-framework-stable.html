  • Built-in XML Frameworks (Document Types)

    The XML frameworks that are supported in Oxygen XML Author

    • DocBook 4 Document Type (Framework)

      A document type standard for books, articles, and other prose documents (particularly technical documentation)

    • DocBook 5 Document Type (Framework)

      An enhanced (version 5) document type standard designed for a variety of documents (particularly technical documentation)

    • DocBook 5.1 Assembly

      A document type standard used to define the hierarchy and relationships for a collection of DocBook resources

    • DocBook 5.1 Topic

      A document type standard used as a modular unit of DocBook documentation

    • DocBook Targetset Document Type (Framework)

      A document type standard used to resolve cross references with the DocBook olink

    • DITA Topics Document Type (Framework)

      An XML-based architecture designed for authoring, publishing, and delivering technical information

    • DITA Map Document Type (Framework)

      A document type that collects and organizes references to DITA topics or other maps

    • XHTML Document Type (Framework)

      Extensible HyperText Markup Language includes the same depth of expression as HTML, but also conforms to XML syntax

    • TEI P5 Document Type (Framework)

      A standard for the academic community that collectively defines an XML format for text that is primarily semantic rather than presentational

      • TEI P5 Author Mode Actions
      • Customization of TEI Frameworks Using the Latest Sources
      • Customization of TEI Frameworks Using the Compiled Sources
    • TEI ODD Document Type (Framework)

      An XML-conformant specification that allows you to create TEI P5 schema in a literate programming style

    • jTEI Document Type (Framework)

      A highly restrictive customization (only about 80 elements are included) of the TEI P5 standard

    • JATS Document Type (Framework)

      A technical standard that defines an XML format for scientific literature

    • EPUB Document Type (Framework)

      An e-book file format that is a ZIP archive and can be downloaded and read on devices such as phones, tablets, computers, or e-readers

    • Other Supported Document Types

      A list of other supported document types with limited support (including document templates)

Customization of TEI Frameworks Using the Compiled Sources

The following procedure describes how to update to the latest stable version of TEI Schema and TEI XSL, already integrated in the TEI framework for Oxygen XML Author.

  1. Go to https://code.google.com/p/oxygen-tei/;
  2. Go to Downloads;
  3. Download the latest uploaded .zip file;
  4. Unpack the .zip file and copy its content in the Oxygen XML Author frameworks folder.
