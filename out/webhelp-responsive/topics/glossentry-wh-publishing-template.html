Oxygen Publishing Template

Oxygen Publishing Template defines all the aspects related with the look and feel(layout and styles) for the WebHelp Responsive output.

The template is self-contained and packed as a ZIP archive making it easy t share with others. It represents the main method for customizing the WebHelp Responsive output.
