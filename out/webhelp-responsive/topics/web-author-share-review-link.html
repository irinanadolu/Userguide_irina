Sharing a Link to a Document for Review

To collect feedback on a particular document (for example, feedback from a subject matter expert), you can copy the URL in the address bar of your browser and share it with the reviewer.

Sharing the Context DITA Map

If you share a DITA documents that uses keys, you may want to make sure that your collaborator will see the keys resolved to the same content. There are several approaches to ensure this:
  • Instruct your collaborator to choose the correct map using the Show DITA Map view button on the DITA toolbar.
  • Use the Impose DITA Map plugin to automatically impose the correct DITA map so that you will not have to construct the URL by hand before sharing it.
  • Generate Oxygen XML WebHelp with edit links and share the link to the generated HTML page. There will be an Edit Online link near each topic title that has the correct DITA map configured.
  • You can share the URL to the DITA map directly and when opened in the editor, the DITA map contains references to topics.
  • Manually construct a shareable URL that includes the DITA map pre-set and share that instead.
