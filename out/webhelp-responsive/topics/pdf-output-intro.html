DITA to PDF Output Customization

Oxygen XML Author provides support for generating PDF output using transformation scenarios for certain types of documents (for example, DITA, DocBook, TEI, and JATS) and Oxygen XML Author supports several different types of processors. There are numerous ways to customize the published output to fit your specific needs.
