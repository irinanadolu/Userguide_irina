    Feature comparison between Oxygen XML Web Author and Oxygen XML Editor/Author

Feature Matrix

Oxygen XML Web Author was designed to provide the functionality of the Author mode in Oxygen XML Editor/Author in a mobile-friendly platform and can be embedded in a third-party standalone Java application to provide WYSIWYG-like XML editing and reviewing directly in your choice of web browsers on your mobile devices or desktops.
